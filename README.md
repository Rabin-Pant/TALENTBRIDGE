# TalentBridge 🌉

A full-stack Job Portal Web Application built with React, Node.js/Express, PostgreSQL and Prisma.

## Tech Stack
- **Frontend:** React 18, Vite, Tailwind CSS, React Router v6
- **Backend:** Node.js, Express.js, JWT Auth
- **Database:** PostgreSQL + Prisma ORM

## Getting Started

### Backend
cd server
npm install
npx prisma migrate dev
npm run dev

### Frontend
cd client
npm install
npm run dev

## Roles
- **Seeker** — Browse jobs, apply, track applications
- **Employer** — Post jobs, manage applicants
- **Admin** — Platform-wide management
